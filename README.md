# Django REST API

## Description
This project is a backend REST API built with Django and Django REST Framework.  
It provides core functionality for modern web applications, including authentication, data management, and scalable architecture.

---

## Features
- JWT Authentication (login, refresh tokens)
- RESTful API endpoints
- User management system
- Database integration with PostgreSQL
- Modular and scalable project structure

---

## Tech Stack
- Python
- Django
- Django REST Framework
- PostgreSQL
- Docker

---
